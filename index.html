HyperText Bilang kit

Introduction


Hyperlink-bilang-kit aka (Hbk) is a python library which is made for the front-end. This framework is a combination of both HTML and CSS. So,the development which requires HTML and CSS can be eased by learning this perticular framework. This would save time for the developer in the front-end works.it's open and free to use by anyone.
